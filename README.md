Mini Kanban Board is a task management web application built with Next.js 15, TypeScript, and Tailwind CSS. It allows users to organize tasks across three stages — Pending, In Progress, and Completed.
Features include:

Create, edit, and delete tasks with confirmation
Move cards across columns using arrow buttons or drag & drop
Search and filter cards in real time
Data persistence using localStorage (cards stay after page refresh)
Loading and empty states for better user experience
Fully responsive design for mobile and desktop
